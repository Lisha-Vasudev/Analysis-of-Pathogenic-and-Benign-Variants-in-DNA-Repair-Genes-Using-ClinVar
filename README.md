# Variant Analysis in DNA Repair Genes Using ClinVar

## Project Overview

This repository contains a computational analysis of pathogenic and benign genetic variants in clinically important DNA repair genes. The study uses curated variant data from ClinVar to systematically examine how variant characteristics—including genomic location, variant type, functional consequence, and allele frequency—distinguish pathogenic from benign variants.

**Key Question:** How are pathogenic and benign variants distributed across DNA repair genes in terms of genomic location, variant type, and functional consequence, and what patterns systematically distinguish pathogenic variants from benign variants?

---

## Biological Motivation

DNA repair pathways are essential for maintaining genome stability and preventing disease. Variants in DNA repair genes are implicated in cancer susceptibility, hereditary cancer syndromes, and other genetic diseases. This analysis focuses on a curated set of 19 genes representing major repair mechanisms:

- **Mismatch Repair (MMR):** MLH1, MSH2, MSH6, PMS2
- **Homologous Recombination (HR):** BRCA1, BRCA2, RAD51, XRCC3
- **Base Excision Repair (BER):** APE1, XRCC1
- **Nucleotide Excision Repair (NER):** XPA, XPC, ERCC1
- **Checkpoint & Tumor Suppressors:** TP53, CHEK2, ATM, PTEN
- **DNA Polymerase Fidelity:** POLE, POLD1

By analyzing curated variant data, this project demonstrates how genomic variation patterns can be explored to gain biological insight without requiring computationally intensive sequencing pipelines.

---

## Dataset

**Source:** ClinVar (NCBI/NIH)
- **File:** `variant_summary.txt` (publicly available at: https://ftp.ncbi.nlm.nih.gov/pub/clinvar/)
- **Filtering strategy:**
  1. Restrict to 19 DNA repair genes (listed above)
  2. Keep only variants with clinical significance = "Pathogenic" or "Benign"
  3. Retain variants with quality review status (classified by multiple submitters or higher)
  4. Remove duplicate entries

- **Expected size:** ~5,000–15,000 variants across ~19 genes
- **Processed data size:** ~3–5 MB (CSV format)

**Important limitations:** This analysis is based on curated ClinVar annotations and does not incorporate population-scale frequency data from external sources (e.g., gnomAD) or experimental validation studies. Allele frequency analysis is conditional on data availability in the ClinVar variant_summary.txt file.

## Methods Summary

### Preprocessing & Quality Control (Notebook 1)
1. Download and parse ClinVar variant_summary.txt
2. Filter to DNA repair genes and clinical significance categories
3. Validate genomic coordinates and remove duplicates
4. Inspect data completeness and distribution
5. Generate processed dataset for analysis

### Descriptive Statistics & Visualization (Notebook 2)
**Core analyses:**
- **Gene distribution:** Variant counts per gene; pathogenic-to-benign ratio per gene
- **Variant type distribution:** Count and compare variant types (Insertion, Deletion, Substitution, Complex)
- **Functional consequence:** Map variants to consequence classes (missense, frameshift, splice-site, synonymous, etc.)
- **Clinical significance:** Overall breakdown of pathogenic vs. benign variants
- **Allele frequency (conditional):** If available, compare allele frequency spectrum between pathogenic and benign variants

**Interpretation:** Results are connected to known biology of DNA repair genes and discussed in the context of cancer genomics literature.

## Setup & Execution

### Requirements
- Python 3.8+
- See `requirements.txt` for package versions

### Installation

1. **Clone or download this repository**
   ```bash
   git clone <repository-url>
   cd variant-analysis-dna-repair
   ```

2. **Create a virtual environment (recommended)**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

### Run the Analysis

**Option A: Run notebooks sequentially (interactive)**
1. Start Jupyter
   ```bash
   jupyter notebook
   ```
2. Open and run `notebooks/01_data_preparation.ipynb`
3. Open and run `notebooks/02_analysis_and_interpretation.ipynb`

**Option B: Run with orchestration script (automated)**
```bash
python run_analysis.py
```

This will execute both notebooks in order and save results to `results/`.

## Results Summary

- Final dataset size: 42,854 variants
- Genes analyzed: 55 (including multi-gene annotations)
- Clinical significance distribution:
  - Pathogenic: 84.7%
  - Benign: 15.3%

### Key Observations:
- Variants are highly enriched in DNA repair genes such as BRCA1, BRCA2, and ATM
- Deletions and single nucleotide variants are the most common variant types
- Pathogenic variants are more frequent than benign variants, reflecting known ClinVar bias
- Some variants map to multiple genes, influencing gene-level distribution and ratio calculations

These results are consistent with known biological roles of DNA repair genes in disease, particularly cancer susceptibility.

## Key Findings & Interpretation

The analysis identifies:
1. **Gene-level patterns** — Which DNA repair genes harbor the highest variant burden; pathogenic enrichment by gene
2. **Variant type associations** — Differences in variant types between pathogenic and benign variants
3. **Functional consequence patterns** — Which consequence types are most predictive of pathogenic classification
4. **Allele frequency patterns** — If available, lower allele frequencies in pathogenic variants (expected)

Results are interpreted in the context of:
- Known DNA repair gene function and cancer biology literature
- ClinVar community consensus on pathogenicity
- Genomic distribution and evolutionary conservation

## Limitations

1. **Curated annotations only** — Analysis relies on ClinVar community consensus; no independent experimental validation
2. **No population frequency integration** — External allele frequency databases (gnomAD, etc.) are not merged
3. **Selection bias in ClinVar** — Variants in ClinVar are not a random sample; overrepresentation of disease-associated variants
4. **Clinical significance classification** — "Uncertain significance" variants excluded; binary comparison may oversimplify
5. **Allele frequency completeness** — Allele frequency fields may be sparse; conditional analysis only if sufficient data
6. **No transcript isoform mapping** — Variants mapped to genes, not specific isoforms; domain-level analysis not included (extension)
7. **No regulatory annotation** — Does not incorporate experimental enhancer/promoter data
8. **Some variants are associated with multiple gene annotations in ClinVar, which may affect gene-level summaries.
